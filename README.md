# -Traffic-Accidents-Analysis-Project-
•	Analyzed a real-world traffic accidents dataset (2,09,000+ records) using Python to identify key patterns in crash severity, weather, lighting conditions, and temporal trends.
•	Performed comprehensive EDA using Pandas, NumPy, Matplotlib, and Seaborn to detect correlations, outliers, missing values, and trend behaviors.
•	Generated meaningful insights and visualizations to support data-driven recommendations for improving road safety and understanding accident causes
